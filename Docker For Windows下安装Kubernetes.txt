
Docker For Windows下安装Kubernetes

1-Docker for Windows的安装
下载地址：https://docs.docker.com/docker-for-windows/install/

2-配置镜像源：在Docker -> Settings ->Daemon中，配置 registry mirror 为 https://registry.docker-cn.com

3-通过chocolatey安装kubectl
（1）安装chocolatey
    a-这是基于.NET Framework 4以上的windows安装软件的命令行工具
    b-管理员方式运行powershell,执行如下【Get-ExecutionPolicy】，如果返回Restricted 执行下一步
    c-执行【Set-ExecutionPolicy AllSigned】 或者 【Set-ExecutionPolicy Bypass -Scope Process】
    d-执行【iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))】
    e-执行【choco】，测试是否安装成功，如需升级，执行【choco upgrade chocolatey】
（2）安装kubectl
    a-执行【choco install kubernetes-cli】
    b-执行【kubectl version】测试是否成功

4-为了加快安装，使用阿里云批处理，提前把k8s需要的images(镜像)拉下来
 （1）执行克隆【git clone https://github.com/AliyunContainerService/k8s-for-docker-desktop.git cd k8s-for-docker-desktop】
 （2）执行【load_images.ps1】，将镜像导入docker
 （3）执行【docker images】查看是否导入成功
 
5-打开Docker for Windows -> Settings -> Kubernetes,勾选开启项进行安装
